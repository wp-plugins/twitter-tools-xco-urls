Use the X.co URL Shortener with the Twitter Tools plugin. This plugin is directly based off the Bit.ly plugin that is bundled with Twitter Tools.

== Installation ==
1. Install the Twitter Tools Plugin
1. Download the .zip file
2. Unpack into wp-content/plugins/ folder
3. Activate the plugin
4. In the Twitter Tools settings, include your X.co API Key

or

1. Login into your wp-admin section
2. From the Plugins menu choose 'Add New'
3. Search for 'Better Adsense Targeting'
4. Click on the 'Install' link for this plugin
